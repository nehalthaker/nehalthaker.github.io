Theft Detection Using Data Science

The paper describes the intruder detection system,which comprises the Atmega-8 microcontroller that is used as the central processing unit of the system and the data from the reed switch and vibration sensors will be communicated to the cloud using an ESP 32 WIFI module. Here we have utilized state of the art SVM ( Support Vector Machine) model for classification which sends the result in the form of an alert through the application. The system uses IoT to detect an anomaly, which refers to the network of connected objects that can communicate and exchange data among themselves. It is used for various applications in the safety and surveillance industry. When it comes to security our house safety is as important as ours. The paper proposes a low cost, flexible, compact and low maintenance intruder detection system. As the intruder detection system is fully automated, once installed it does not require any kind of human interactions. The main aim is to make a better and cheaper version of the system.
